# 华英兽药智能推荐系统 - 部署指南

## 部署方式一：Linux服务器部署

### 环境要求
- Linux服务器（Ubuntu 18.04+/CentOS 7+/Debian 9+）
- Python 3.9+
- 至少 2GB 内存
- 开放端口 8501

### 部署步骤

1. **上传部署包**
   ```bash
   # 使用 scp 上传部署包到服务器
   scp -r deploy_package root@服务器IP:/opt/
   ```

2. **登录服务器**
   ```bash
   ssh root@服务器IP
   ```

3. **进入部署目录**
   ```bash
   cd /opt/deploy_package
   ```

4. **运行部署脚本**
   ```bash
   chmod +x deploy.sh
   ./deploy.sh
   ```

5. **访问应用**
   ```
   http://服务器IP:8501
   ```

### 手动部署步骤

1. **安装依赖**
   ```bash
   cd /opt/deploy_package
   pip install -r requirements.txt
   ```

2. **启动应用**
   ```bash
   # 前台运行（测试用）
   streamlit run app_mobile.py --server.port 8501 --server.address 0.0.0.0
   
   # 后台运行（生产环境）
   nohup streamlit run app_mobile.py --server.port 8501 --server.address 0.0.0.0 > /var/log/huaying-app.log 2>&1 &
   ```

3. **配置服务（systemd）**
   ```bash
   cat > /etc/systemd/system/huaying-vet-app.service << EOF
   [Unit]
   Description=华英兽药智能推荐系统
   After=network.target

   [Service]
   Type=simple
   WorkingDirectory=/opt/deploy_package
   ExecStart=/usr/bin/python3 -m streamlit run app_mobile.py --server.port 8501 --server.address 0.0.0.0
   Restart=always
   RestartSec=5
   User=root

   [Install]
   WantedBy=multi-user.target
   EOF

   systemctl daemon-reload
   systemctl enable huaying-vet-app
   systemctl start huaying-vet-app
   ```

### 常用命令

```bash
# 查看服务状态
systemctl status huaying-vet-app

# 启动服务
systemctl start huaying-vet-app

# 停止服务
systemctl stop huaying-vet-app

# 重启服务
systemctl restart huaying-vet-app

# 查看日志
journalctl -u huaying-vet-app -f
```

## 部署方式二：Streamlit Cloud 部署

### 部署步骤

1. **访问 Streamlit Cloud**
   - 打开浏览器，访问：https://share.streamlit.io
   - 使用您的 GitHub 账号登录

2. **创建新应用**
   - 点击页面右上角的 "New app" 按钮
   - 填写以下信息：
     - **Repository**: 选择您的GitHub仓库
     - **Branch**: 选择 `main`
     - **Main file path**: 输入 `app_mobile.py`

3. **点击 "Deploy" 按钮**

4. **等待部署完成**
   - 首次部署可能需要 2-5 分钟
   - 部署完成后，您会看到一个访问链接

### 访问应用

部署完成后，您可以：
- 在电脑浏览器访问
- 在手机浏览器访问
- 将链接分享给其他人使用

## 部署方式三：Windows服务器部署

### 环境要求
- Windows Server 2016+
- Python 3.9+
- 至少 2GB 内存

### 部署步骤

1. **复制部署包到服务器**
   - 将 `deploy_package` 文件夹复制到 `C:\huaying-vet-app`

2. **运行部署脚本**
   ```cmd
   cd C:\huaying-vet-app
   deploy.bat
   ```

3. **访问应用**
   ```
   http://localhost:8501
   ```
