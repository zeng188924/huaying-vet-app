# 华英兽药推荐系统 - Streamlit Cloud 部署指南

## 部署步骤

1. **访问 Streamlit Cloud**
   - 打开浏览器，访问：https://share.streamlit.io
   - 使用您的 GitHub 账号登录

2. **创建新应用**
   - 点击页面右上角的 "New app" 按钮
   - 填写以下信息：
     - **Repository**: 选择 `zeng188924/huaying-vet-app`
     - **Branch**: 选择 `main`
     - **Main file path**: 输入 `app_mobile.py`

3. **点击 "Deploy" 按钮**

4. **等待部署完成**
   - 首次部署可能需要 2-5 分钟
   - 部署完成后，您会看到一个访问链接
   - 链接格式：`https://zeng188924-huaying-vet-app.streamlit.app`

## 访问应用

部署完成后，您可以：
- 在电脑浏览器访问
- 在手机浏览器访问
- 将链接分享给其他人使用

## 注意事项

- 应用是公开的，任何人都可以访问
- 如果长时间无访问，应用会进入休眠状态
- 下次访问时会自动唤醒（可能需要等待 30 秒）

## 更新代码

如果您需要更新代码：
1. 在 GitHub 上修改代码
2. Streamlit Cloud 会自动检测并重新部署
