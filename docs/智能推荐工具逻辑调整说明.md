# 智能推荐工具逻辑调整说明

**文档版本**：V1.0  
**调整时间**：2026-07-02  
**调整范围**：`drug_recommendation_system_full.py` 智能推荐算法  

---

## 一、调整背景

根据产品数据清理要求，系统已移除所有标记为 **"明星产品"**（`source=明星产品`）的数据记录，仅保留同时属于 **"底价目录"** 和 **"产品信息-华英"** 分类体系的产品数据。清理后：

- 底价目录产品：**21** 个
- 产品信息_华英产品：**74** 个（含名称中包含明星品牌名但 `source=产品信息_华英` 的正常产品）
- 合计可用产品：**95** 个

原推荐算法在产品加载和匹配打分中存在对明星产品的识别逻辑与潜在加权倾向。为适应清理后的产品结构，消除品牌偏向，确保底价目录产品与华英产品公平竞争，需对推荐算法逻辑进行同步调整。

---

## 二、调整目标

1. **移除明星产品相关逻辑**：清理推荐引擎中对明星产品的识别、加权、兜底等处理；
2. **适应产品分类体系变化**：推荐逻辑基于新的 `source` 体系（`底价目录`、`产品信息_华英`）运行；
3. **优化推荐权重计算**：以适应症匹配为核心，疾病类型和用途场景为辅助，来源完整度为微调；
4. **保证推荐稳定性与响应速度**：通过功能测试与性能测试验证系统运行稳定。

---

```text
推荐逻辑调整：
- 取消对明星产品的额外加权，确保底价目录与华英产品公平竞争
- 以适应症匹配为核心权重，疾病类型与用途场景作为辅助权重
- 华英产品信息来源给予少量信息完整度加成
```

### 3.2 匹配分数计算逻辑调整

调整 `_calculate_match_score` 方法，新的权重设计如下：

| 打分维度 | 权重 | 说明 |
|----------|------|------|
| 基础分 | 1.0 | 所有参与计算的产品初始分 |
| 适应症匹配 | +3.0 / 命中 | 核心权重，直接决定推荐相关性 |
| 疾病类型命中 | +1.0 | 产品 `disease_types` 包含用户疾病类型时加分 |
| 用途场景匹配 | +0.5 | 预防倾向中药/免疫增强剂/维生素/抗病毒类；治疗倾向抗生素/化药 |
| 来源完整度加成 | +0.3 | `source = "产品信息_华英"` 时给予小幅加成 |
| 明星产品加权 | **已移除** | 不再对任何品牌设置额外加分 |

**核心变化**：

- 取消明星产品额外加权，避免推荐结果向特定品牌倾斜；
- 底价目录产品不再因缺少明星标签而处于劣势，完全依靠适应症和用途匹配获得排名；
- 华英产品因信息字段相对完整，获得轻微来源加成，但不改变以适应症为核心的排序逻辑。

### 3.3 单药推荐选择逻辑调整

`_calculate_single_recommendations` 方法调整要点：

1. **基于统一匹配分数排序**：所有产品按 `_calculate_match_score` 得分倒序排列；
2. **保留适应症真正匹配的产品**：过滤后仅保留适应症命中或匹配分数 ≥ 3.0 的产品；
3. **兜底策略简化**：当无匹配产品时，优先返回相关类型（如细菌性药物用于生殖系统疾病）且通过产蛋期安全过滤的药物；
4. **去重机制**：按 `drug.id` 去重，确保最终返回最多 3 个不重复产品。

### 3.4 组合推荐逻辑调整

组合推荐方案已全面切换为清理后可用产品：

- 呼吸道方案：底价目录化药 + 华英呼吸道产品；
- 肠道方案：底价目录抗菌药 + 华英肠道调理产品；
- 抗病毒方案：华英抗病毒/中药类产品；
- 抗球虫方案：底价目录磺胺类 + 华英中药抗球虫产品；
- 产蛋期安全方案：仅选用 `egg_period_safe = true` 的产品。

所有组合方案均经过配伍禁忌检测模块校验，确保安全提示准确。

## 四、算法伪代码

```python
def _calculate_match_score(drug, diseases, request, disease_type):
    score = 1.0

    # 1. 适应症匹配（核心）
    for indication in drug.indications:
        for disease in diseases:
            if disease in indication or indication in disease:
                score += 3.0

    # 2. 疾病类型命中
    if disease_type in drug.disease_types:
        score += 1.0

    # 3. 用途场景匹配
    if request.usage == "预防":
        if drug.category in ['中药', '免疫增强剂', '维生素', '抗病毒类产品']:
            score += 0.5
    else:  # 治疗
        if drug.category in ['抗生素', '化药']:
            score += 0.5

    # 4. 来源信息完整度微调
    if drug.source == "产品信息_华英":
        score += 0.3

    return score
```
