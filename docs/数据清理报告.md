## 一、清理目标与规则

### 1.1 清理目标

根据业务要求，对产品数据进行全面筛选与清理，确保推荐系统仅使用符合规范的产品信息。

### 1.2 保留规则

清理后的产品数据必须**同时满足**以下两个条件：

1. 产品分类属性明确属于 **"底价目录"** 分类；
2. 产品信息归类包含在 **"产品信息-华英"** 分类下。

实际操作中，以数据记录 `source` 字段为准，仅保留 `source = "底价目录"` 或 `source = "产品信息_华英"` 的记录。

### 1.3 删除规则

- 彻底移除所有标记为 **"明星产品"** 的文件及相关数据记录（`source = "明星产品"`）；
- 删除关联的元数据、关联表记录及文件存储；
- 清理过程中不影响系统其他功能模块的正常运行，不删除非产品相关的基础配置数据；
- **注意**：名称中包含明星品牌名但 `source` 为 `产品信息_华英` 的记录不属于明星产品，应保留。
